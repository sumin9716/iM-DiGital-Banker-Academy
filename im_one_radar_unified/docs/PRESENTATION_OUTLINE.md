# 발표 구성(권장 10장 내외)

1. 배경/문제: 실적 리포트 중심 사후 대응, 세그먼트별 변동성 확대
2. 목표: 예측-조기경보-설명-추천을 한 흐름으로 통합, FX 레짐 확장
3. 사용자/의사결정 단위: 마케팅/CRM, RM, FX 담당 / 세그먼트 단위
4. 데이터/세그먼트 설계: 내부 KPI + 외부(달력/금리/환율) / 세그먼트 키
5. KPI 정의: 메인 5 + FX 2, 파생 KPI(한도소진율, 디지털비중 등)
6. 엔진 아키텍처: Forecasting / Risk / Explain / NBA(룰+스코어)
7. 산출물: 대시보드 4탭 + 월간 1-page 리포트
8. 평가/백테스트: 롤링 백테스트, Hit Rate@TopN, Precision/Recall@TopN, uplift
9. FX 특화: 레짐(추세×변동성) + Opportunity/Risk score + 국면 기반 액션
10. 로드맵/확장: TFT/계층 예측/변화점 탐지/캠페인 반응 기반 추천 고도화

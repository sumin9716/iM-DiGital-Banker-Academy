#### 슬라이드 3: 현업의 Pain Point

> **핵심 메시지**: 현업 담당자가 겪는 4가지 문제점 강조

| 문제 | 상세 설명 |
|------|----------|
| ❌ 사후 대응 체계 | 월간 실적 확정 후 분석, 이상징후 사전 감지 어려움 |
| ❌ 비효율적 분석 | 2,000+ 세그먼트 일일이 분석, 우선순위 정하기 어려움 |
| ❌ 원인 파악 지연 | 변화 발생해도 원인(드라이버) 파악에 시간 소요 |
| ❌ 대응 타이밍 상실 | 선제적 대응 불가, 최적의 대응 시점 놓침 |

```

│     → 2,000+ 세그먼트 일일이 분석                │

│                                                 │

│                                                 │
│  ❌ 대응 타이밍 상실                             │
```

```
            ▼
     [출력: 1~3개월 예측]
```
